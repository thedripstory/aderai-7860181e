# Aderai - Klaviyo Segment Automation

**70+ Pre-Built Klaviyo Segments in 30 Seconds**

Free SaaS tool that simplifies Klaviyo email segmentation for e-commerce brands. Deploy agency-grade customer segments instantly.

## Features

- 70+ Pre-Built Segments across 5 categories
- One-Click Deployment to Klaviyo
- AI-Powered Segment Suggestions
- Analytics Dashboard
- Mobile Responsive
- Dark Mode Support

## Tech Stack

- React 18, TypeScript, Tailwind CSS
- Supabase (Database, Auth, Edge Functions)
- Resend API for emails
- OpenAI for AI features

## Quick Start

```bash
npm install
npm run dev
```
